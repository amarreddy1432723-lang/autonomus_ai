# Feature Guide

## Terminal

The terminal lives at the bottom like VS Code/Cursor. Electron local terminal is primary for trusted folders; backend runtime terminal is fallback.

## Editor

Monaco powers editing. The roadmap includes diagnostics, hover docs, symbols, references, rename, and inline autocomplete.

## Preview

Preview verification runs a browser check, captures screenshots, console errors, network failures, HTML snapshots, and blank-page signals.

## GitHub PR Flow

GitHub App is the production path:

1. Connect/install app.
2. Choose repo.
3. Import files.
4. Create branch.
5. Commit approved changes.
6. Open PR.
7. Poll checks.

## Apps And Plugins

Plugins can add agent skills, panels, or tools. The v1 registry validates manifests and stores approved permissions.
